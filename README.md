# 정처기코

## 개요

정처기코는 정보처리기사 프로그래밍 문제에 대한 솔루션을 제공하는 서비스입니다.

링크 : [https://jungcheogico-omega.vercel.app/](https://jungcheogico-omega.vercel.app/)

## 기술

FRONTEND

- Next.js
- React.js
- TailwindCSS
- Zustand

BACKEND

- Next.js Serverless Function
- MongoDB

DEPLOY

- Vercel

MONITORING

- Vercel Logs
- Google Analytics

## 기능

- 언어별 문제 조회
- 문제 솔루션 조회
- 문제 즐겨찾기
- 댓글 조회/작성/삭제
